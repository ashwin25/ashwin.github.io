AWS DEVOPS - Lets Get Hands on !!
Lets spin up an EC2 in our VPC using AWS Cloudformation!
AWS CLI Command to spin up a new cf stack 

	aws cloudformation create-stack --stack-name ashwinteststack 
	--template-body file://ec2_isntance_with_security_group.txt 
	--parameters ParameterKey=KeyName,ParameterValue=ashwin_admin_keyMay2020
Lets wait for cf to do its work !
Command to keep waiting till resources are created in AWS 

	aws cloudformation wait stack-create-complete --stack-name ashwinteststack
Lets check logs in AWS Cloudformation!
Spit the logs! 

	aws cloudformation describe-stack-events --stack-name ashwinteststack
Lets check what it created in AWS Cloudformation!
Spit the output ! 

	aws cloudformation describe-stack-events --stack-name ashwinteststack
Let's Update our stack in AWS Cloudformation!
Update to allow port 80 on the security group 

	aws cloudformation update-stack --stack-name ashwinteststack 
	--template-body file://ec2_isntance_with_security_group.txt 
	--parameters ParameterKey=KeyName,ParameterValue=ashwin_admin_keyMay2020
Let's Tear down our AWS Cloudformation Stack!
First delete-stack and then wait for it to tear down. 

	aws cloudformation delete-stack --stack-name ashwinteststack
	aws cloudformation wait stack-delete-complete --stack-name ashwinteststack
